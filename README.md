# AI Voice Assistant

A real-time Voice AI Assistant built using

- Streamlit
- Whisper
- Groq
- Llama 3.3 70B

## Features

✅ Voice Input
✅ Whisper Speech Recognition
✅ AI Conversation Memory
✅ Text-to-Speech
✅ Download Chat
✅ Animated UI

## Installation

pip install -r requirements.txt

streamlit run app.py
